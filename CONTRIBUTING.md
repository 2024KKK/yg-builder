# Contributing to Topspeed Builder

感谢你参与 Topspeed Builder。欢迎提交问题、改进建议和代码贡献。

## 开发环境

- Node.js 22+
- npm
- Windows、macOS 或 Linux

```bash
npm install
npm run typecheck
npm run build
```

## 提交修改

1. 从 `main` 创建一个功能分支。
2. 保持每个提交聚焦于一个问题或功能。
3. 提交前运行 `npm run typecheck` 和 `npm run build`。
4. 提交 Pull Request，并说明修改内容、验证方式和可能的兼容性影响。

## API Key 与本地数据

不要提交 API Key、个人项目目录、生成素材、构建产物或包含隐私的截图。API Key 只应通过应用设置在本机配置。
